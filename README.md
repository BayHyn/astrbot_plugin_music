## 🤝 介绍

音乐搜索、热评

## 📦 安装

- 直接在astrbot的插件市场搜索astrbot_plugin_music，点击安装，等待完成即可

- 也可以克隆源码到插件文件夹：

```bash
# 克隆仓库到插件目录
cd /AstrBot/data/plugins
git clone https://github.com/Zhalslar/astrbot_plugin_music

# 控制台重启AstrBot
```

## ⌨️ 配置

请前往插件配置面板进行配置

## 使用说明

|     命令      |      说明       |
|:-------------:|:-----------------------------:|
| /点歌 歌名      | 根据序号点歌,可以附加歌手名  |
